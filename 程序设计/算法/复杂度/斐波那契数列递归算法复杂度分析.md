# 斐波那契数列(Fibonacci)递归算法复杂度分析

## 斐波那契数列

斐波那契数列格式： `F(n)=F(n-1)+F(n-2)` <br/>
**1, 1, 2, 3, 5, 8, 13, 21, 34, ……**

javascript实现：

``` javascript
const fibonacci = n => {
    if (n == 0 || n == 1) return n;
    return fibonacci(n - 1) + fibonacci(n - 2);
}

console.log(fibonacci(6)) //8
console.log(fibonacci(7)) //13
```

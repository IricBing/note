# nvm 基础用法篇

## 查看远端所有镜像版本

``` shell
$ nvm ls-remote
```

## 安装指定版本node

``` shell
$ nvm install 12.18.2
```

## 使用安装好的版本

``` shell
$ nvm use node
```

## 查看node版本

``` shell
$ nvm run node --version
```

## 附

* 指定淘宝镜像源安装`nrm`

``` shell
$ npm i nrm -g --registry=http://registry.npm.taobao.org
```

# Ubuntu 时区设置

## 查看当前系统时间

``` shell
$ Thu, 24 Dec 2020 11:00:03 +0800
```

可以看到现在的时区是**东八区**（ `+0800` ），有的时候可能是UTC时间，后面显示+0000，这个时候我们就要调整时间了。

``` shell
$ sudo cp /usr/share/zoneinfo/Asia/Shanghai  /etc/localtime
```

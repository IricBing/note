# Postgresql 运算符优先级

一般情况下，级别高的运算符先进行计算，如果级别相同，按表达式顺序从左到右依次计算。

优先级顺序：

``` mermaid
graph TD;
  A[最低] --> B["=(赋值运算符), :="];
  B --> C["| OR"];
  C --> D["AND"]
  D --> E["NOT"]
  E --> F["BETWEEN, CASE, WHEN, THEN, ELSE"]
  F --> G["=(比较运算符), >=, >, <=, <, <>, !=, IS, LIKE, IN"]
  G --> H["-(减), +"]
  H --> I["*, /, %"]
  I --> J["-(负号)"]
  J --> K["!"]
  K --> L["最高"]
```

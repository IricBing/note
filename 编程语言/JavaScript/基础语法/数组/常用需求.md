# JavaScript数组常用需求

## 数组去重

**简单数据：**

``` javascript
const arr = [1, 2, 3, 2, 4, 3, 3, 2, 5];
console.log([...new Set(arr)]);
//打印结果 [1, 2, 3, 4, 5]
```

**复杂数据：**

``` javascript
const productList = [{
    name: 'a',
    category: '衬衫'
}, {
    name: 'b',
    category: '卫衣'
}, {
    name: 'd',
    category: '衬衫'
}, {
    name: 'f',
    category: '短袖'
}, {
    name: 'e',
    category: '短袖'
}]
console.log([...new Set(productList.map(product => product.category))]);
// 打印结果 ["衬衫", "卫衣", "短袖"]
```

## number 数组求和

``` javascript
const arr = [1, 2, 3, 2, 4, 3, 3, 2, 5];
console.log(arr.reduce((a, b) => a + b));
// 打印结果 25
```
